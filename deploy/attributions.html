this webspace would not exist as it does without plethora of online resources. one can find much of the scaffolding of its foundations here.

code.

css.

98.css by jdan; the code which was used as a base for these foundational structures. CSS resets via josh w. comeau.

the CRT effect was created by aleclownes; that is, most of it, as the way which they mimicked the scanlines was quite rough on the processors, thus [we] decided to create something for personal usage that meets the effect in a less resource-consuming way.

javascript.

preloader by iamwebrock. site statistics are displayed using max's javascript code. tab-toggling functionality by fikri-code.

tooltips created using tippy.js.

assets.

images.

all pokémon assets are sourced from spriters-resource; namely:

  • pokémon firered / leafgreen,
  • pokémon mystery dungeon: red rescue team,
  • pokémon diamond / pearl,
  • pokémon platinum,
  • pokémon heartgold / soulsilver,
  • and pokémon mystery dungeon: explorers of sky.

custom cursors by bg5df, albeit modified as to match the sites theme. most images featuring celestial objects are sourced from NASA.

images are dithered using this tool. original windows 98 icons sourced from this website by alexmeub.

background image pattern reference was sourced from stocksy and made into a simplistic monotone circuit-board pattern by the webmaster.

some images have been sourced from pinterest and the OS cannot accurately say where the image is originally from. images of unknown source will be used sparsely.

audio.

all sounds played (i.e. upon the click of a button) are sound effects from pokémon emerald version, sourced from sounds-resource.

fonts.

alkhemikal by jeti.

awfully digital by aryel filipe.

dogica pixel by roberto mocci.

kiwi soda by jeti.

modern DOS 8x16 by jayvee enaguas.
